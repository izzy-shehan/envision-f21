Attributes example

The "a" HTML element stands for anchor, and is how we can create hyperlinks to other pages within our site, or other external pages entirely.

Here is an anchor element without any attributes. It's just plain text!

Using the "href" attribute (hypertext reference), we can specify where the link should direct us to when clicked.
The image tag (img) requires a similar attribute to function properly.

Below is a blank image tag.

And here is an image tag with the "src" argument, which stands for the source of the image (and is very similar to href...)

The img tag has even more attributes we can set, though.
